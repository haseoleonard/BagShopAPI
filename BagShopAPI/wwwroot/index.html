/api/products

/api/products GET get all Products
/api/products/{id} GET get selected Products by ID
/api/products POST Create new Product
    Form properties:
  • productName - text
  • description - text
  • price - number
  • quantity - number
  • categoryID - number
/api/products PUT Update Product
    Form properties:
  • productID - number - hidden
  • productName - text
  • description - text
  • price - number
  • quantity - number
  • categoryID - number
